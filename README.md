# Seven-Segment

In the seven segment display that exists on digital screens, 7 LED lights are placed as shown below, which can display different numbers from 0 to 9, and to display a k digit number, k of these 7 segments must be used.


![Webp net-resizeimage](https://user-images.githubusercontent.com/118338875/224101750-8014b5bd-a307-4ce2-9b98-e35d2f0ccf26.jpg)

In number printing, we know that each segment is divided into two parts with three lines (top, middle and bottom lines).

The input consists of only one line where two natural numbers n and b are separated by a space.

1 <= n <= 10^18 & 1 <=b<= 20
